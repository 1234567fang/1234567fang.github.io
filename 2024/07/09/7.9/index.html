电脑的选择

1、笔记本
2、台式机
性价比转化率=0.6 （台式6k=笔记本1w）
品牌：联想 华硕 华为 小米 火影 七彩虹
主要不同在于：售后服务
主板：
决定有多少usb接口，雷电4=usb3.2>usb3.1>usb3.0。建议最少3个，优选4个
2. 看有多少m.2硬盘位(3个会很舒服)(预装硬盘没必要，溢价太严重了，京东2t固态7000的才1k多)
3. 看散热，看看就行了，毕竟
4. 不建议水冷笔记本
显卡：
N卡稳定
A卡更便宜
CPU：
优先I卡，i7性价比高
i3 i5 i7后面数字是指损毁率，越高损毁率低
硬盘：
建议1T和4T，性价比高
内存条：
包括8 16 32 64 96 128
看主频率DDR（CPU决定上限）

硬件-驱动-系统-用户
window版本 家庭 专业 企业 专业工作站 教育
服务器版本 2003 2008 2012 2018

永恒之黑的漏洞CVE-2020-0796

#github全球最大的代码开源网站
github上下载相关代码
复制粘贴放到kali中
kali与win10开启NAT模式
关闭win10hei的杀毒（win防火墙和病毒保护），查找win10和kali的ip地址
#poc观点验证脚本
#exp漏洞验证脚本

// -p windows/x64/meterpreter/bind_tcp 设置payload为windows64位meterpreter控制台下的反向连接

// -b ‘\x00’ 设置在生成的程序中避免出现’\x00’
